## Provision Azure infrastructure for the ZavaStorefront web application

Provision Azure infrastructure for the ZavaStorefront web application with the following requirements:

## Infrastructure Components
- Linux App Service for hosting the web application
- Azure Container Registry for Docker image storage
- Application Insights for monitoring
- Azure AI Foundry for access to GPT-4 and Phi models
- All resources in westus3 region (where models are available)

## Requirements
- Use Docker for deployment to App Service
- No local Docker installation required
- App Service should use Azure RBAC for pulling images from Container Registry (not passwords)
- All resources deployed together in a single resource group
- Same region for all resources (westus3)
- Development environment

## Tools & Approach
- Use Azure Developer CLI (AZD)
- Infrastructure as Code with Bicep
- Single deployment for all resources
